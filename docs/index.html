Unicorn Surprise

Licorne dansante & Oiia Cat a chaque lancement d'app

Comment ca marche ?

🔄
A chaque demarrage ou deverrouillage, une fenetre vous demande si vous voulez activer ou desactiver les animations.
🦄
Quand vous lancez une application, une licorne danse pendant 4 secondes avec des effets arc-en-ciel.
🐱
Puis l'Oiia Cat recule en mode "oiia oiia" pendant 3 secondes. Total : 7 secondes de fun.
🛡️
Consentement renouvele : le choix ne dure que jusqu'au prochain demarrage/deverrouillage. Vous restez toujours en controle.
⚠️ Ce programme s'installe sur votre ordinateur et se lance a chaque demarrage. Il detecte les ouvertures d'applications pour afficher des animations. Vous pouvez le desactiver a tout moment ou le desinstaller.

🐧 Installation Linux

  1. 1 Telechargez le projet : git clone https://github.com/OWNER/unicorn-surprise.git
  2. 2 Lancez l'installeur : cd unicorn-surprise && bash install_linux.sh
  3. 3 C'est pret ! L'app se lancera au prochain demarrage.

🍎 Installation macOS

  1. 1 Telechargez le projet : git clone https://github.com/OWNER/unicorn-surprise.git
  2. 2 Lancez l'installeur : cd unicorn-surprise && bash install_mac.sh
  3. 3 C'est pret ! L'app se lancera a chaque connexion.

🪩 Installation Windows

  1. 1 Telechargez le projet : git clone https://github.com/OWNER/unicorn-surprise.git
  2. 2 Ouvrez PowerShell en admin et lancez : cd unicorn-surprise; powershell -ExecutionPolicy Bypass -File install_windows.ps1
  3. 3 C'est pret ! L'app se lancera a chaque demarrage.
